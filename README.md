# Adeptus Craftmatica

> A creative command center for miniature painters, hobbyists, collectors, and campaign organizers.

Adeptus Craftmatica is a cross-platform hobby management application designed to bring structure, momentum, and satisfaction to creative workflows.

Track projects. Organize paints. Manage armies. Plan campaigns. Log hobby sessions. Build collections. Visualize progress.

Built by a hobbyist, for hobbyists.

---

### 📊 Dashboard System
- Smart overview dashboard
- Recommended actions
- Agenda + reminders
- Quick action system
- Live project summaries

### 📅 Calendar & Scheduling
- Month / Week / Day / Agenda views
- Event management

- Timeline tracking for hobby progress

- Milestones with progress tracking

## 🧠 Philosophy

Adeptus Craftmatica is designed around one core idea:

Creative hobbies deserve professional-grade tools.

Most hobby apps focus on isolated features. Adeptus Craftmatica focuses on workflow — helping creators stay organized, motivated, and connected to their projects over long periods of time.

The goal is not just tracking collections.

The goal is helping people finish things.
